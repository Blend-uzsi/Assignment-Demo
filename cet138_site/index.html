What is this site?

This site serves as my CET138 (Full Stack Development) ePortfolio submission. It presents working demonstrations with annotated code, evidencing competency in semantic HTML, responsive CSS, Bootstrap 5, client-side JavaScript, and foundational full-stack concepts including the HTTP request–response cycle, API consumption, and deployment.

For Bootstrap components to load, you need an internet connection (CDN). For JS “fetch”, we simulate a server so it works from a local file.
